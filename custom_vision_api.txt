Custom Cog API

How to use the Prediction API
If you have an image URL:


https://southcentralus.api.cognitive.microsoft.com/customvision/v1.0/Prediction/2f3ab445-617c-42c6-a5b6-d73009d8db56/url?iterationId=e0ab8df5-f72b-4c01-b3d6-761866f8cec7
Set Prediction-Key Header to : 6c9f96aa9a574ab8b3aa18f5e0ec4c1c
Set Content-Type Header to : application/json
Set Body to : {"Url": "<image url>"}
If you have an image file:


https://southcentralus.api.cognitive.microsoft.com/customvision/v1.0/Prediction/2f3ab445-617c-42c6-a5b6-d73009d8db56/image?iterationId=e0ab8df5-f72b-4c01-b3d6-761866f8cec7
Set Prediction-Key Header to : 6c9f96aa9a574ab8b3aa18f5e0ec4c1c
Set Content-Type Header to : application/octet-stream
Set Body to : <image file>
Remember, you can mark an iteration as Default so you can send data to it without specifying an iteration id. You can then change which iteration your app is pointing to without having to update your app.
